# Vika 1, 12.–18. janúar 2026

## Vikan

Fyrri partur fyrirlesturs á miðvikudegi fer í

- kynningu á námskeiði,
- yfirferð á námsefni,
- praktískt mál.

Í seinni part förum við yfir verkefni 1 og byrjum að leysa það saman.

Mikilvægastu verkefni nemenda verða að setja upp umhverfi (node, npm, github, netlify) og sækja grunn að verkefni.

Í næstu viku höldum við áfram að leysa verkefni 1 saman í samhengi við námsefni.

## Fyrirlestrar

Koma inn eftir fyrirlestur 14. janúar.

## Verkefni

- [ ] Setja upp node.js útgáfu 24 (og þar með npm) með [`nvm`](https://github.com/nvm-sh/nvm) eða [`nvm-windows`](https://github.com/coreybutler/nvm-windows).
- [ ] Setja upp git og GitHub account, ef þarf
  - [ ] Mæli með að setja upp með [SSH lykli](https://docs.github.com/en/authentication/connecting-to-github-with-ssh).
- [ ] Útbúa Netlify aðgang, ef þarf.
- [ ] Visual Studio Code með a.m.k. extensionum: `EditorConfig for VS Code`, `ESLint`, `Prettier`.
- [ ] Fara fyrir node.js námsefni, sérstaklega `async-await` dæmi.
- [ ] Fara yfir eininga námsefni, sérstaklega `es-module` dæmi.
- [ ] Byrja á [verkefni 1](https://github.com/vefforritun/vef2-2026-v1).
- [ ] Skoða [verkefni 1 frá 2025](https://github.com/vefforritun/vef2-2025-v1) ([sýnilausn](https://github.com/vefforritun/vef2-2025-v1-synilausn)) og [verkefni 1 frá 2024](https://github.com/vefforritun/vef2-2024-v1) ([sýnilausn](https://github.com/vefforritun/vef2-2024-v1-synilausn)).
